С 04.09.2023г. ООО «Башкирэнерго» предоставляет дополнительные платные услуги по технологическому присоединению энергопринимающих устройств граждан к электрическим сетям. Данной услугой могут воспользоваться заявители, объекты присоединения которых соответствуют следующим критериям - объект присоединения классом напряжения до $0 { , } 4 \ \mathrm { \kappa B }$ по III категории надежности: заявители физические лица – до $1 5 ~ \mathrm { \bf { \kappa } \mathrm { { B T } } }$ ; заявители ИП и юридические лица – до $6 0 ~ \mathrm { \kappa B T }$ ; присоединение выполняется воздушным вводом проводом СИП; присоединение осуществляется непосредственно к электрическим сетям ООО «Башкирэнерго». Согласно нормам действующего законодательства, в отношении вышеперечисленных категорий заявителей в обязательствах сетевой организации не предусмотрено выполнение мероприятий по фактическому присоединению. Вместе с условиями типового договора об осуществлении технологического присоединения в личном кабинете заявителя будет размещена утвержденная инструкция, содержащая последовательный перечень мероприятий, обеспечивающих безопасное осуществление фактического присоединения и фактического приема напряжения и мощности.
